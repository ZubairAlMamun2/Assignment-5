Important FAQ

What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents, representing structure.

How do you select an element form the DOM?

Use methods like `getElementById`, `querySelector`, or `getElementsByClassName`.

What is event delegation in the context of the DOM, and why useful?

Event delegation allows handling events at a parent level, improving performance and simplifying event management.

How do you manipualte an elment's attributes and styles using the DOM?

Use `setAttribute` for attributes and `style.property` for inline styles to manipulate elements easily.
